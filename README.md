# 📊 Portfolio Optimization using Markowitz & CAPM

## Overview

This project applies quantitative finance techniques to construct optimal portfolios using:

* Markowitz Mean-Variance Optimization
* Capital Asset Pricing Model (CAPM)
* Monte Carlo Simulations
* Efficient Frontier Analysis

## Key Insights

* Achieved a Sharpe Ratio of ~1.49 using an international portfolio
* Demonstrated diversification benefits with NVDA
* Compared CAPM vs Markowitz under real market conditions

## Tech Stack

* Python (NumPy, Pandas, Matplotlib)
* Financial Modeling

## Results

* Domestic Sharpe Ratio: 1.44
* International Sharpe Ratio: 1.49
* NIFTY50 Sharpe Ratio: -0.07
